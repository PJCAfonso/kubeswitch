# Kubernetes Version Switcher 

![Go](https://github.com/steamhaus/kubeswitch/workflows/Go/badge.svg?branch=master)

## Easily switch kubectl binary versions.

### Build/Run:

```
$ go build -o kubeswitch .

$ chmod +x kubeswitch 

$ mv kubeswitch $PATH

```
